## WARNINGS

1. **mermaid v11 float precision**: The deterministic test was relaxed from
   byte-exact comparison to structural comparison + SHA-256 verification because
   mermaid's constraint-based layout engine (dagre) produces slightly different
   floating-point path coordinates on each run. The SHA-256 of the Mermaid source
   (not SVG output) remains fully deterministic.

2. **npm install disk impact**: `node_modules/` is ~176MB (mermaid 11.16.0 alone
   is 83MB, 1171 files). This is a one-time install cost and the standard mermaid
   footprint. node_modules is gitignored.

3. **JSDOM getBBox stub**: The render script uses a stub `getBBox()` returning
   fixed {x:0, y:0, width:100, height:20} because JSDOM doesn't implement SVG
   text measurement. This is sufficient for mermaid's layout engine which only
   needs approximate text dimensions.
